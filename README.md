Сцена Gameplay
Перед сборкой можно настроить параметры игры через инспектор в GameConfig в папке Assets
Управление: W, A, D + мышь
Стрельба: пробел или ЛКМ
После появления астероид не может нанести игроку урон в течение 2 секунд (время можно изменить)
В роли жизней выступает щит (правый верхний угол), который уменьшается после каждого столкновения. После исчезновения щита столкновение с астероидом уничтожает корабль
